## Computing

[[edit](/w/index.php?title=Info&action=edit&section=1 "Edit section: Computing")]

* [.info](/wiki/.info ".info"), a generic top-level domain
* [info:](/wiki/Info: "Info:"), a URI scheme for information assets with identifiers in public namespaces
* [info (Unix)](/wiki/Info_(Unix) "Info (Unix)"), a command used to view documentation produced by GNU Texinfo
* [Info.com](/wiki/Info.com "Info.com"), a search engine aggregator
* .info, the filename extension for metadata files used by the [Amiga Workbench](/wiki/Workbench_(AmigaOS) "Workbench (AmigaOS)")
* [.nfo](/wiki/.nfo ".nfo"), a filename extension for informational text files accompanying compressed software.
